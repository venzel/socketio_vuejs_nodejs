# Socketio com VueJs

## Recursos utilizados

- Socket.io
- JWT para validação da comunicação
- Middleware para barrar uma comunicação inválida
- Criação de uma biblioteca com padrão de projeto observer

## Rodar o VueJs

```bash
cd frontend-vue

yarn serve
```

## Rodar a API do NodeJs

```bash
cd api-node

node index.js
```
